# ToeDoe

A simple barcode scanner app built in Swift 4 in iOS 11. Scan the barcode of the product
with smartphone or tablet to get information about the products ingredients.

## Built With

*  Swift 4.0
*  Xcode 9.0
*  Alamofire 4.7
*  SwiftyJSON 4.1
*  ChameleonFramework 2.1
